# Market Easy App

MarketEasy is a app for listing products of a store, just sign in and check out the list.

## Installation

### Fork it!

Install dependencies

```
$ yarn
```

### Android

```
$ yarn android
```

### IOS

```
$ cd ios
$ pod install
$ yarn ios
```
